Users table

http method path what it does
get /users returns users table as an array of objects
post /users adds a new user to the users table
get /users/:id returns a single user from the table as an object in an array
patch /users/:id edits a single user
delete /users/:id deletes a single user
post /users/login allows a user to login after checking if username and passwords match
